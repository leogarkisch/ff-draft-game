A snake draft is a drafting system where the pick order reverses each round, creating a "snake-like" pattern that ensures fairness.

How it Works:

  • Round 1: Teams pick in order 1, 2, 3, ..., {{ num_teams }}
  • Round 2: Teams pick in reverse order {{ num_teams }}, {{ num_teams - 1 }}, ..., 2, 1
  • Round 3: Back to normal order 1, 2, 3, ..., {{ num_teams }}
  • Round 4: Reverse again {{ num_teams }}, {{ num_teams - 1 }}, ..., 2, 1

Why Use a Snake Draft?

The snake draft system balances the advantage of picking first. While the team with pick #1 gets the best player overall, they also get the worst pick in round 2. This creates more balanced teams and makes the draft more exciting for everyone!
